# Basel III Bond Demand Visualizer

A BondStats tool to visualize how Basel III-style liquidity assumptions may influence structural demand for government bonds.

## Features

- Estimate liquidity buffer size
- Estimate HQLA stock
- Estimate government bond demand
- Scenario sensitivity analysis
- Breakdown and interpretation
- Responsive design

## Files

- `index.html`
- `style.css`
- `script.js`

## Deployment

This project is designed for GitHub Pages.

## Disclaimer

This is a simplified analytical and educational tool. It is not a regulatory reporting model.
